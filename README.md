## About
An image based on [suchja/wine](https://hub.docker.com/r/suchja/wine/) providing you the latest version of Sparx Systems's [Enterprise Architect Lite](http://www.sparxsystems.com/products/ea/index.html). This image provides the *Lite* Version of *Enterprise Architect*, which can only read UML-Models created with another version.
